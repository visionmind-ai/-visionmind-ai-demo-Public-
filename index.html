Frequently Asked Questions

Everything you need to know about VisionMind AI OS

How does the licensing work?
We offer two license types: Developer License ($10,000) for individual developers or small teams, and Enterprise License ($25,000) for organizations requiring premium support and customization. Both are one-time payments with no recurring fees.
What kind of support do you provide?
Developer License includes community support via Discord. Enterprise License includes priority support with SLA, dedicated account manager, and custom feature development options.
Is the source code included?
Yes, both licenses include full source code access via private GitHub repository. You can modify, extend, and deploy the code in your own infrastructure.
What are the system requirements?
VisionMind AI OS requires Python 3.8+, 8GB RAM minimum (16GB recommended), and support for Docker. It can run on Windows, Linux, or macOS, and is cloud-agnostic.
Do you offer refunds?
Due to the digital nature of our product and immediate access to source code, we do not offer refunds. We encourage you to try the public demo and request a technical walkthrough before purchasing.
